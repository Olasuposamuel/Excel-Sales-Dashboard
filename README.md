# Online-Store-Data-Analysis (Interactive Dashboard design using MS Excel)
## Project objective
To analyze the sales performance for Online Store to uncover key business insights such as revenue trends, product performance and customer behavior to improve sales.

## Dataset used
- <a href="https://github.com/Olasuposamuel/Excel-Sales-Dashboard/blob/main/Samuel%20Data%20Analysis(Excel).xlsx"> Dataset</a>

## Questions(KPIs)
- Compare the sales and orders using single chart.
- Which month has the highest sales and orders?
- Status with the most order.
- Who purchased more - Men or Women?
- List top 5 states contributing to the sales.
- Relation between age and gender based on number of orders.
- Highest selling category.
 
- Dashboard Interaction <a href="https://github.com/Olasuposamuel/Excel-Sales-Dashboard/blob/main/Screenshot%20Online%20Store%20Dashboard.png"> view Dashboard</a>

## Process
- Cleaned and organized raw data
- Removed duplicates and handled missing values
- Created pivot tables for the questions asked.
- Built KPIs such as Total Sales and Total Order ID 
- Designed an interactive dashboard for visualization
